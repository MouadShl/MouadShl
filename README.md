## 🚀 Featured Project — PFE @ Ziwig Morocco

> **Endometriosis Patient Insights — Full Data Science Pipeline**  
> End-to-end pipeline from raw web data to executive Power BI dashboards

| Stage | Description | Tools |
|-------|-------------|-------|
| **E1** | Multi-source web scraping (forums, associations) | Python, BeautifulSoup, Requests |
| **E2** | Data cleaning & normalization | Pandas, regex, custom rules |
| **E3** | NLP preprocessing & language detection | spaCy, langdetect |
| **E4** | Sentiment analysis (3 models) | CamemBERT, RoBERTa, XLM-RoBERTa |
| **E5** | KPI calculation & scoring | Pandas, SQL Server |
| **E6** | Topic modeling | BERTopic, HDBSCAN, UMAP |
| **E7** | Star-schema Data Warehouse | SQL Server, T-SQL |
| **BI** | Interactive dashboards (2 axes) | Power BI |
